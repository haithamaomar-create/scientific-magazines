# التوثيق

## نظرة عامة

موقع المجلات العلمية لكلية السلام الجامعة هو نظام متكامل لإدارة وعرض المجلات العلمية.

## هيكل المشروع

راجع [README.md](../README.md) للاطلاع على هيكل المشروع الكامل.

## واجهة برمجة التطبيقات (API)

### نقاط النهاية المتاحة

#### فحص الصحة
- **GET** `/api/health` - التحقق من أن الخادم يعمل

#### المجلات
- **GET** `/api/magazines` - الحصول على جميع المجلات
- **GET** `/api/magazines/:id` - الحصول على مجلة محددة

## المتغيرات البيئية

راجع [.env.example](../.env.example) للاطلاع على المتغيرات البيئية المطلوبة.
